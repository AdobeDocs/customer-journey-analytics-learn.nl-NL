# [!UICONTROL line] visualisaties toevoegen aan [!DNL Analysis Workspace] projecten in [!DNL Customer Journey Analytics]

Leer [!UICONTROL line] visualisaties toevoegen en configureren in [!DNL Analysis Workspace] -projecten in [!DNL Customer Journey Analytics] . Deze visualisatie vertegenwoordigt metriek gebruikend een lijn om te tonen hoe de waarden in tijd veranderen.

>[!NOTE]
