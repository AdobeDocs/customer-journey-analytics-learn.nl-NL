# Verbeterde rapportage voor Adobe Journey Optimizer

In Adobe Journey Optimizer zijn nieuwe functies beschikbaar voor rapportage en orkestmetingen. Deze verbeteringen voor Journey Optimizer omvatten ook meer interoperabiliteit en een grotere waarde voor Customer Journey Analytics via extra gegevensbronnen, geavanceerde aanpassingen, gegevensmanipulatie, analyse en meer.
